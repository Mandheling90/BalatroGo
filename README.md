# 화투록 MVP

같은 월의 화투 네 장을 맞춰 점수패를 모으고 부적으로 규칙을 강화하는 싱글 플레이 웹게임입니다.

```bash
npm install
npm run dev
```

## 게임 방법

- 각 앤티는 스몰 블라인드, 빅 블라인드, 보스 블라인드 순서로 진행됩니다.
- 스몰과 빅 블라인드는 보상을 포기하고 패스할 수 있지만, 보스는 패스할 수 없습니다.
- 매 턴 손패에서 한 장을 골라 바닥에 놓습니다.
- 차례를 넘길 때마다 덱에서 두 장이 바닥에 자동으로 펼쳐집니다.
- 손패로 바닥의 같은 월을 맞추면 낸 패와 짝을 점수패로 가져옵니다.
- 같은 월 네 장이 바닥에 모두 모이면 네 장을 한꺼번에 가져옵니다.
- 획득한 광·열끗·띠·피와 고도리·홍단·청단·초단으로 점수를 계산합니다.
- 10턴 안에 목표 점수를 달성하면 장터에서 부적을 구매하고 다음 판으로 갑니다.
